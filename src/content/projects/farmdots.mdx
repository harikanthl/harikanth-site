FarmDots visualizes farm operations on interactive maps — field boundaries, sensor points, and operational history in one view.

## Technical approach

- **MapLibre GL** for performant vector rendering
- **PMTiles** for cheap, static tile hosting
- **Cloudflare Workers** for edge API and auth

## Vision

A digital twin that helps farmers see patterns across seasons — not just dots on a map, but actionable spatial intelligence.
